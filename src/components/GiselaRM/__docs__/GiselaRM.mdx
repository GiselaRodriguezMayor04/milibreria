# GiselaRM

Componente que muestra una tarjeta con un avatar, un título y una descripción.

#### Ejemplo

<Canvas of={GiselaRM.Primary} />

## Uso

```tsx
import GiselaRM from 'milibreria';

function Example() {
  return (
    <GiselaRM
      title="Gisela Romero"
      description="A Creative Mind"
      avatar="https://example.com/gisela-avatar.jpg"
    />
  );
}

export default Example;
